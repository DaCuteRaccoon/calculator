# Running
If you don't use [Replit](https://replit.com/~), just do the following command:
```sh
cargo run
```
If you have any errors, make sure you have EVERYTHING installed.
- rustc
- rustfmt
- cargo
- cargo-edit
- rust-analyzer

Try running it again. It should work.
